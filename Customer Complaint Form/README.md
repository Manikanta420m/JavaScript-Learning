Customer Complaint Form

A simple and user-friendly web application that allows customers to submit complaints and feedback online. The form collects customer details, complaint information, and stores the submitted data for review and resolution.

Features :
            Customer name, email, and contact number input
            Complaint category selection
            Detailed complaint description
            Form validation for required fields
            Responsive design for desktop and mobile devices
            Success message after submission
            Easy-to-use interface

Technologies Used :
                    HTML5
                    CSS3
                    JavaScript (ES6)
