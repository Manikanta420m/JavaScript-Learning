# 🎵 Music Player

A simple and responsive Music Player built with **HTML**, **CSS**, and **JavaScript**. This project allows users to play, pause, skip tracks, go back to previous songs, and automatically play the next song when the current one ends.

## 🚀 Features

* ▶️ Play songs
* ⏸️ Pause songs
* ⏭️ Next song
* ⏮️ Previous song
* 🔄 Automatically plays the next song after the current song ends
* 🎵 Displays album artwork, song title, and artist
* 📃 Playlist support
* 📱 Responsive and user-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## ▶️ Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/music-player.git
```

2. Navigate to the project folder:

```bash
cd music-player
```

3. Open `index.html` in your browser.

No additional installation or dependencies are required.

## 📖 How It Works

* Click **Play** to start playback.
* Use **Next** and **Previous** buttons to navigate between songs.
* The player updates the album artwork and song information automatically.
* When a song finishes, the next song starts playing automatically.
